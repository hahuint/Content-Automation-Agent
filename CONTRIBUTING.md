# Contributing to Content Automation Agent

First off, thank you for considering contributing! It's people like you who make this a great tool for the community.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* Use a clear and descriptive title.
* Describe the exact steps which reproduce the problem.

### Suggesting Enhancements
* Open an issue describing the feature you'd like to see.
* Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Install the dev dependencies: `make setup`.
3. If you've added code that should be tested, add tests.
4. If you've changed APIs, update the documentation.
5. Ensure the test suite passes: `make test`.
6. Use clear, descriptive commit messages.

## Development Environment
* Python 3.9+
* [Ollama](https://ollama.com/) running on your host.
* Docker (optional but recommended).

## Style Guide
We use `ruff` for linting. You can check your code style with `make lint`.
